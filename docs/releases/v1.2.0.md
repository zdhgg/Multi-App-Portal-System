# v1.2.0 - 控制中心界面焕新与运维体验升级

发布日期：2026-03-24

## Highlights

- 门户首页、应用管理、PM2 管理、端口管理、系统设置统一升级为控制中心式布局，关键指标、快捷操作和状态摘要更集中。
- 应用卡片与详情弹窗强化了技术栈、访问端口、运行模式、运行时间、访问入口和最近更新等核心信息展示。
- PM2 进程表格、端口占用列表和用户资料卡改为统一的视觉风格，异常进程与冲突端口更易识别。
- 引入新的全局设计变量、排版和响应式间距，前端整体层级、可读性和一致性显著提升。

## Included Work

- `Portal.vue` 重构为 Hero + Metrics + Content 的首页控制台结构。
- `Management.vue`、`PM2Management.vue`、`PortManagement.vue`、`SystemSettings.vue` 完成整页视觉与信息架构升级。
- `AppCard.vue` 与 `AppDetailDialog.vue` 提升应用信息密度，离线应用默认动作调整为更安全的“查看详情”路径。
- `PM2ProcessTable.vue`、`PM2StatsPanel.vue`、`PortManager.vue` 优化状态映射、异常提示和运维操作入口。
- `style.css` 新增全局字体、色板、表面层级、阴影和动效降级变量，为多页面统一视觉系统提供基础。

## Compatibility

- 本次为语义化版本 `1.2.0` 的前端体验增强发布。
- 未引入已知的破坏性配置格式变更。

## Verification
