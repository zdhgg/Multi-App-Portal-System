# v1.2.1 - 门户交互与卡片一致性修复补丁

发布日期：2026-03-25

## Highlights

- 修复门户首页用户资料相关弹窗在特殊布局与裁剪上下文下可能被错误定位到页面下方的问题。
- 调整离线应用卡片操作区，保留“查看详情”作为主操作，并将次级操作改为直接可点击的“刷新”按钮。
- 为单卡片刷新增加请求中禁点与 2 秒冷却，减少重复点击和重复请求。
- 修复 `video-cms` 等单行标题卡片在标题高度和底部操作区上的视觉不一致问题。

## Included Work

- `UserProfile.vue` 与 `LoginDialog.vue` 的弹窗改为挂载到 `body`，避免受门户首页特殊布局影响。
- `AppCard.vue` 重构离线态操作区，离线卡片改为“查看详情 + 刷新”，在线卡片继续使用更多菜单。
- `Portal.vue` 增加单卡片刷新状态与冷却管理，补充成功/失败提示。
- 同步根项目、前端、后端、配置模板、脚本头部和发布文档中的版本标识到 `1.2.1`。

## Compatibility

- 本次为语义化版本 `1.2.1` 的补丁发布。
- 未引入已知的破坏性接口或配置格式变更。

## Verification

- 已执行 `npm --prefix main-portal run build:check`
