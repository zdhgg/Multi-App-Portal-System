# v1.3.0 - 备份中心与运维控制台增强

发布日期：2026-03-27

## Highlights

- 新增系统备份与恢复中心，统一管理配置快照、文件归档、恢复和清理流程。
- 新增自动备份调度器，支持从系统设置实时读取周期、时间、保留天数和输出路径。
- 运维页面继续统一为状态页头与快照指标布局，并补强 `external-exe` 应用配置同步链路。

## Included Work

- `detection-api/src/services/backupScheduler.ts` 新增自动备份调度器，并在系统设置保存和 API 启动/关闭时完成初始化与刷新。
- `detection-api/src/services/configurationExporter.ts` 扩展备份列表、旧版归档兼容恢复、配置快照可用性校验与恢复前备份逻辑。
- `main-portal/src/components/settings/BackupRestorePanel.vue` 与 `main-portal/src/views/SystemSettings.vue` 新增统一的备份策略编辑和备份中心界面。
- `main-portal/src/components/layout/OperationPageHeader.vue` 被复用于 PM2 管理、端口管理等页面，统一运维信息表达。
- `main-portal/src/components/AppConfiguration.vue`、`main-portal/src/services/appConfigApi.ts` 与后端应用更新链路同步增强，避免配置保存后主记录残留旧目录或旧运行目标。
- 同步根项目、前端、后端、系统配置、脚本模板和主要发布文档中的版本标识到 `1.3.0`。

## Compatibility

- 本次为语义化版本 `1.3.0` 的小版本发布。
- 兼容已有 `backups/backup-registry.json` 和旧版归档路径记录，不需要手工迁移历史备份。
- 未引入已知的破坏性认证、路由或应用管理接口变更。
